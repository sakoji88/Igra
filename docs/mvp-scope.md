# MVP scope

MVP покрывает фундамент цикла сезона: авторизация, роли, обзор сезона, поле, базовый поток назначения рана, инвентарь, event log, страницы игроков, rules page, admin/judge controls и legacy import templates.

Вне MVP оставлены интеграции, realtime, публичная регистрация, мобильный клиент и полная оцифровка всех исторических исключений из старого Excel.
