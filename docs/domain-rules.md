# Domain rules

## Fixed rules

- Turn = movement by the sum of two d6.
- Condition choice happens only after movement and destination cell resolution.
- Debuffs override buffs.
- Conflicting items/effects annihilate each other.
- No effect stacking.
- One player may have only one active assignment at a time.
- Judge/admin resolves disputed cases and can rollback safe recent actions.
- Win/Drop are the only player-facing completion buttons.
- Scoring is always server-side.

## Encoded MVP behavior

- Genre conditions use a 1.5 score multiplier.
- Passing start grants a small lap reward.
- Random cells are represented as seeded data-driven entries that can be extended without rewriting page components.
